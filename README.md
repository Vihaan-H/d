# Liquid Glass Bento Dashboard

A single-page bento dashboard with:

- Animated liquid-glass visual style
- Live clock/date
- Weather (via geolocation + Open-Meteo)
- News feed (Hacker News front page)
- Quick links
- Notes saved in localStorage
- Inactivity screensaver with rotating quote and animated non-static backgrounds

## Run

Open `index.html` directly, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit <http://localhost:8080>.
